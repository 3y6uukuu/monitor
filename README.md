## Requirements
```
Nodejs >=7.6.0
```

## Installation
```
npm install -g serve
```

```
cd server/
npm install
```

```
cd client/
npm install
```

## Usage
```
cd server/
npm run start
```

```
cd client/
npm run build
serve -s build
```

## Development
```
cd client/
npm run start
```

## Configs
```
server/src/config.js
client/src/config/index.js
```
